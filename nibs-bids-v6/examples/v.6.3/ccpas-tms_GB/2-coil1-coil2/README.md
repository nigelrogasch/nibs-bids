# ccPAS — element_id = Coil_1|Coil_2

One `nibs.tsv` row. The two coils are written as a `|`-delimited `element_id` (`Coil_1|Coil_2`), with
two ordinary `ElementSet` entries. The delimiter is aligned to the two pulses of `pattern1`
(`Coil_1` fires first, `Coil_2` 8 ms later). See the parent folder README for how this differs from the
other two encodings.
